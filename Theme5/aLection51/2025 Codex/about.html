Мета системи

Mini-IAS показує, як навіть один структурований JSON може стати джерелом ситуаційної обізнаності для штабу. Користувач отримує індикатори сил, деталізацію за підрозділами, інтерактивну карту та історичні тренди.

Структура sample.json

  • enemyStrength — агреговані сили (особовий склад, техніка, моральний стан).
  • geo.units/activity_points/routes/clusters — геопросторові об'єкти та активність.
  • dailyTrends — історія застосування UAV, артилерії та вильотів.
  • threatAssessment — градація секторів за рівнями загроз.
  • notes — вільний текст аналітиків.

Olive Military UI

Палітра вибудувана навколо кольорів #3B5323, #556B2F та #C7D3A9. Заголовки використовують шрифт Saira Stencil One для формування відчуття тактичного інтерфейсу, а панелі мають жорстку геометрію, наголошуючи на службовому характері ІАС.

Джерела та документація

• Дані: навчальний файл sample.json.
• Карта: Leaflet.js + leaflet.heat.
• Графіки: Chart.js.
• Шрифти: Google Fonts (Saira Stencil One, Source Sans 3).
